# GreenEvent AI

AI-Powered Sustainable Event Planning Assistant

## Overview

GreenEvent AI helps event organizers estimate resource consumption and improve sustainability using Artificial Intelligence.

## Features

- Sustainability Score Calculation
- Food Requirement Estimation
- Water Consumption Estimation
- Plastic Waste Estimation
- Electricity Usage Estimation
- AI-Powered Sustainability Recommendations using Google Gemini

## SDGs Supported

- SDG 11: Sustainable Cities and Communities
- SDG 12: Responsible Consumption and Production
- SDG 13: Climate Action

## Technologies Used

- Python
- Flask
- HTML
- CSS
- Google Gemini AI
